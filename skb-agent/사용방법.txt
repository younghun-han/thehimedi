[SKB 전화 데이터 수집 프로그램 사용 방법]

이 폴더(skb-agent)를 병원에 있는 '전화기가 연결된 PC'나 '카운터 PC'로 복사해 주세요.

1. 준비 사항:
   - 해당 컴퓨터에 'Java (자바)'가 설치되어 있어야 합니다.
   - 없을 경우: 구글에서 '자바 설치'를 검색하여 설치해 주세요. (JDK 설치 권장)

2. 실행 방법:
   - 이 폴더 안에 있는 'run_agent.bat' 파일을 더블 클릭하여 실행합니다.
   - 까만 창이 뜨면서 'Connected to SKB Server!'라는 메시지가 나오면 성공입니다.
   - 이 창을 끄지 마시고, 최소화해서 계속 띄워두시면 됩니다. (24시간 권장)

3. 복사 방법:
   - USB 메모리에 이 폴더(skb-agent) 전체를 담아서 옮기시거나,
   - 압축해서 본인에게 이메일로 보내신 뒤 병원 PC에서 다운로드 받으셔도 됩니다.

4. 확인:
   - 프로그램이 실행 중일 때 전화가 오면, '더하이메디' 웹사이트의 '로그 기록'에 자동으로 기록이 남게 됩니다.
