[더하이메디 SKB 중앙 집중식 매니저 v2.0 - 무설치 버전]

본 패키지는 자바(Java)나 Node.js 설치 없이도 윈도우 컴퓨터에서 바로 실행 가능한 
"제로 디펜던시" 버전입니다.

1. 필수 준비물
   - Windows 운영체제 (7, 10, 11 또는 Server)
   - .NET Framework 4.0 이상 (대부분의 윈도우에 기본 내장)
   - 인터넷 연결 (Supabase DB와 통신용)

2. 파일 목록 (폴더 통째로 복사하세요)
   - SKB_OpenAPI_IMS.dll  (SKB 통신 엔진)
   - SKBAgentNative.cs    (에이전트 소스)
   - SKBManager.cs        (중앙 관리 시스템 소스)
   - run_central_skb.bat  (원클릭 시작 파일)

3. 실행 방법 (이게 끝입니다!)
   - `run_central_skb.bat` 파일을 더블 클릭하여 실행합니다.
   - 프로그램이 자동으로 코드를 빌드하고 DB 감시를 시작합니다.

4. 병원 추가/관리 방법
   - 웹 대시보드(Hospital Settings)에서 병원별 SKB AppKey를 입력하면, 
     중앙 매니저가 이를 실시간으로 감지하여 연결을 시작합니다.

※ 주의사항
   - SKB본사와 직접 통신하므로, 가능하면 병원 인터넷 회선이 있는 곳에서 
     실행하는 것을 권장합니다. (외부에서 실행 시 연결 성공 여부를 확인하세요)
